GOALS

- make the test pipeline in SC1TEST stable 
    - advertise it 
    - make results and coverage as clear as possible

- have a readable draft of the testing strategy
    - define and involve a first working group
    - involve the testing council

- show the framework around

- estimate what tests we need in automated pipelines - and how many - to support a minimum viable SC1 Release
